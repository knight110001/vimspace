# VimSpace

### :sparkles: Description ###
VIM Vundle Packages


### :rocket: Installation ###

```
1. git clone https://github.com/knight110001/vimspace.git
2. mv vimspace .vim
3. cd .vim
4. ./install.sh
5. upgrade vim
sudo add-apt-repository ppa:jonathonf/vim
sudo apt-get update
sudo apt-get install vim
```
